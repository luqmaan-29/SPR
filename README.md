# Speech Processing Labs (SPR)

This repository serves as the digital lab manual for the Speech Processing (SPR) course. It contains a comprehensive collection of experiments, code implementations, and practical exercises designed to provide hands-on experience with speech signal processing concepts.

The materials cover a wide range of topics, from fundamental signal analysis to advanced applications like Speech-to-Text (STT) systems, utilizing tools such as Python, Jupyter Notebooks, and Streamlit.
